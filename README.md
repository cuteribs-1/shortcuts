## 大妈三连
1. 用 Chrome/Edge 登录大妈网站.
2. 按 F12 打开开发工具
3. 顶上选项卡切换到 Application
4. 左边栏选中 Cookies/https://smzdm.com
5. 在右边找到 Name 为 sess 的值 xxxxxxx (长得像随机密码的 base64 字符串)
6. 是后 sess=xxxxxxx; 就是大妈三连需要用的 cookie 了. (注意结果加上 ; )
